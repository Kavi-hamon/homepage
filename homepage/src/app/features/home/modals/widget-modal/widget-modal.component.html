Widget guide for you or your AI

Widgets run in a sandboxed iframe. Use plain HTML, CSS and JS. Avoid real form submission, avoid localStorage, and use the provided storage bridge for persistence.

Supported persistence API
Important rules
  • Do not use a real <form> submit flow. Use button click handlers instead.
  • Keep state JSON-serializable and reasonably small.
  • Links can open in a new tab. Widget state is saved through the homepage app.
  • When logged in, saved widget state syncs through the backend with the rest of the dashboard.
